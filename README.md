# Trading Performance Audit System

Sistem audit performa trading berbasis struktur diagnostik. Dirancang untuk mengubah aktivitas trading menjadi data objektif yang dapat dianalisis dan diulang.

## Core Purpose

Menghilangkan subjektivitas dalam evaluasi trading dengan memisahkan:

* eksekusi vs hasil
* skill vs perilaku
* edge vs noise

## System Structure

Terdiri dari 10 modul utama:

1. Trade Context
2. Execution Audit
3. Psychological State
4. Performance Breakdown
5. Edge Validation
6. Pattern Detection
7. Strength vs Weakness Mapping
8. Precision Correction
9. Final Verdict
10. Export Report

Setiap modul memaksa input spesifik, bukan opini umum.

## Key Features

* Structured tab navigation (HTML single-file)
* Rule-based execution audit
* Cognitive bias checklist (10 bias utama)
* Session scoring (0–10)
* Pattern detection layer
* Auto-generated audit report (.txt export)
* Interactive UI (toggle, checklist, slider)

## Design Philosophy

* No motivational layer
* No hindsight bias
* Focus on repeatable behavior
* Treat trading as a performance system

## Usage

1. Jalankan file HTML
2. Isi setiap modul setelah sesi trading
3. Generate laporan
4. Gunakan hasil untuk iterasi sistem

## Output

Laporan audit berbasis teks:

* objektif
* terstruktur
* siap untuk journaling lanjutan atau analisis statistik

## Positioning

Tool ini bukan jurnal trading biasa.
Ini adalah sistem diagnostik untuk mengidentifikasi:

* titik kebocoran performa
* bias kognitif
* validitas edge
